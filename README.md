FitIndex


FitIndex is a comprehensive mobile health and wellness tracker built with React Native, Expo, and Context API.
It empowers users to monitor their physical health by providing precise Body Mass Index (BMI) calculations, personalized health classifications, 
and a persistent measurement history.
The app features a modern, card-based UI designed for high readability, featuring dynamic feedback messages, real-time data validation,
and a seamless navigation system to track long-term fitness progress and health trends.
